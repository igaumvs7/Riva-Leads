Gerenciar Usuários

Crie e edite os acessos de cada pessoa cadastrada. A senha é convertida em hash antes de salvar — ninguém consegue ver a senha real no Sheets.

Como funciona: selecione a pessoa, defina um nome de usuário e senha. A senha é criptografada (SHA-256) antes de ir para o Sheets — nem você conseguirá ver a senha depois de salvar. Para resetar, basta criar uma nova senha. O campo Ativo controla se o acesso está liberado.
